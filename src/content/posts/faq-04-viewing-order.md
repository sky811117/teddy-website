買方常常會掉進的陷阱:看了 30 幾間房,每一間都覺得「好像可以」「但好像又差一點」,最後無法決定。

通常不是物件的問題,是**沒有 SOP**。每間都「隨意看」,看完就忘,當然分不清差異。

這篇把看屋 3 階段 SOP 寫下來。

---

## 一句話直接答案

看屋分 3 階段,**先後順序不能跳**:

| 階段 | 動作 | 篩出多少 |
|------|------|---------|
| **階段 1** | 線上篩選 + Google Map 掃 | 100 → 20 |
| **階段 2** | 現場初看(白天 + 傍晚 + 雨天) | 20 → 3-5 |
| **階段 3** | 進階驗證(說明書 + 謄本 + 驗屋) | 3-5 → 1 |

**3 個月內完成這 3 階段是正常節奏**。少於 1 個月通常太衝動,超過 6 個月通常猶豫過頭。

---

## 階段 1:線上篩選(在家做,1-2 週)

目的: **從 100+ 間篩到 20 間值得實地看**。

### 用哪些平台

| 平台 | 特色 | 優先度 |
|------|------|--------|
| **591** | 物件最多、更新最快 | ⭐⭐⭐ |
| **樂屋網** | 中部物件比例高 | ⭐⭐⭐ |
| **5168 實價登錄** | 跨品牌比較 | ⭐⭐ |
| **ycut(有巢氏專屬)** | 公司物件,屋主資料較完整 | ⭐⭐ |
| **Facebook 社團** | 屋主自售、議價空間大 | ⭐ |

### 篩選 6 個硬條件

1. **總價範圍**(預算上限往下抓 90%,留議價空間)
2. **區域**(通勤 30 分鐘內、不超過 3 個區)
3. **坪數**(室內坪數,不含車位)
4. **屋齡**(年限上限自訂)
5. **格局**(2 房或 3 房,看家庭規劃)
6. **有無車位**(需求看你)

### Google Map 嫌惡設施掃描

每間候選物件**5 分鐘掃**:

- 搜尋「殯儀館」「公墓」「火葬場」(500 公尺內)
- 搜尋「加油站」「瓦斯」「高壓電」(100 公尺內)
- 搜尋「工廠」「變電所」(200 公尺內)
- 看 3D 視角 + 街景時間軸

**參考**: [Week 04:用 Google Map 揪出 10 大嫌惡設施](/posts/week-04-undesirable-facilities)

掃完還沒被淘汰的 → 進階段 2。

## 階段 2:現場初看(2-4 週)

目的: **從 20 間篩到 3-5 間進階考慮**。

### 一天看幾間?

**最多 4 間,建議 2-3 間**。超過會疲累,每間細節都混淆。

建議的看屋節奏:

```
09:00-09:30  第一間
09:30-10:00  休息 + 寫筆記
10:00-10:30  第二間
10:30-11:00  休息 + 寫筆記
11:00-11:30  第三間
11:30-12:00  總結
```

### 不要帶長輩、不要帶設計師(第一次)

第一次目的是**「我自己喜不喜歡」**。

帶長輩進來 → 長輩會用他們那代的眼光評(風水、座向、財位),你的判斷被覆蓋。
帶設計師進來 → 設計師會想接案子,推你買「他能改造的房」。
帶男朋友/女朋友進來 → 容易吵架。

**第二次或第三次再帶**。

### 看屋要看 3 次(不同時段)

| 時段 | 看什麼 |
|------|--------|
| **白天 10-15:00** | 採光、屋況、坪數感、地段 |
| **傍晚 17-19:00** | 通勤車流、噪音、晚餐時段巷弄狀況 |
| **雨天** | 漏水痕跡、排水、社區排水 |

⚠️ **晚上單獨來一次也很重要** — 看社區治安、停車位有沒有真的好停。

### 進屋 60 秒檢查清單

| 項目 | 看哪裡 |
|------|--------|
| **漏水** | 天花板四角、窗框周圍、廚房水槽下、浴室天花 |
| **壁癌** | 牆面有沒有泡泡、剝落、白色粉狀 |
| **結構裂縫** | 樑柱有沒有 45 度角裂痕(危險),水平/垂直細紋(較安全) |
| **管線** | 開所有水龍頭,看水壓 + 排水 |
| **電** | 開每盞燈、插插頭,看跳電 |
| **氣味** | 進門 30 秒內聞,新油漆味可能蓋臭味 |
| **隔音** | 站在客廳,跟業務聊天,看隔壁有沒有傳音 |

進階參考: 〈漏水屋怎麼看?6 個進屋 60 秒就能發現的訊號〉(下次寫)

## 階段 3:進階驗證(1-2 週)

目的: **從 3-5 間鎖定 1 間下斡旋**。

### 看 3 份文件

1. **不動產說明書** — 仲介依法提供。看凶宅欄、漏水歷史、嫌惡設施
2. **建物所有權狀** / **謄本** — 看坪數、登記項目、有無設定抵押
3. **建物測量成果圖** — 看格局實際範圍,**確認違建有沒有登記**

### 帶 3 個人一起去

第三次或第四次帶這些人:

- **長輩 / 配偶**(意見納入)
- **設計師**(估改造費)
- **驗屋師**(收 3000-8000 元,專業檢測漏水、管線、結構)

⚠️ **驗屋費用屋主不一定願意付,但你買到問題屋的代價遠超這筆**。

### 鄰居打聽 + 房仲圈情報

打去管委會、找隔壁鄰居、附近店家。

「**這社區住起來感覺怎樣?**」 — 一句話可以聽到很多。

## 給你的 action items

下次進場前準備:

1. **建一個 Notion / Excel / Google Sheets** 紀錄每間物件(地址、價格、優缺點、看屋次數)
2. **第一次看屋自己去**,不帶長輩
3. **同一間至少看 3 次**,不同時段
4. **準備 6 個月時間**,不要強迫自己「3 個月內買到」
5. **下斡旋金前一定請看不動產說明書 + 謄本**
6. **預算內留 30 萬給驗屋 + 急修**(萬一驗屋師發現問題)
