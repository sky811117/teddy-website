前陣子一位客戶剛買下北屯一間 28 年的老公寓,30 坪,要翻新。

她興奮地拿著估價單問我:

> 「景泰,工班報 50 萬,這個預算翻 30 坪夠嗎?」

我看了一下:估價單只寫「油漆、地板、衛浴、廚房、油煙機冷氣」。

我問她:「**水電管線換不換?**」
她說:「報價單沒提到耶。」

「**那 28 年的水電管線你打算保留嗎?**」

她愣住。

我跟她說:**最少 80 萬,還要留 10% 預備金,因為老屋拆開來絕對會有意外**。

她後來重新找工班報價,最終預算抓 90 萬,實際做完用了 95 萬(預備金救命)。

中古屋翻新最常踩雷,就是「**預算抓太少**」。這篇給你一份實戰報價單。

---

## 一句話直接答案

**30 坪中古屋翻新標準預算 50-100 萬,簡裝 15-45 萬,高規 100-150 萬以上**。

無論哪個等級,都要**預留 10-15% 預備金**應付拆開後發現的意外。

---

## 三個翻新等級 — 每坪預算對照表

| 等級 | 每坪預算 | 30 坪總價 | 適合對象 |
|------|---------|----------|---------|
| **簡裝** | 5,000-15,000 元 | 15-45 萬 | 投資出租、過渡自住、預算極限 |
| **標準** | 15,000-30,000 元 | 50-100 萬 | 一般自住、想住 10-15 年 |
| **高規** | 30,000-50,000 元 | 100-150 萬+ | 長住、要設計感、進口建材 |

### 簡裝包含什麼

- 全屋油漆(乳膠漆)
- 拋光地板或塑膠地板
- 換衛浴設備(臉盆、馬桶、淋浴)
- 換廚房檯面、烘碗機
- 簡單木工(電視牆、玄關櫃)

### 標準包含什麼

- 簡裝所有項目
- **更換水電管線**(整套換新)
- **更換廚房水槽 + 全套廚具**
- **更換衛浴防水 + 排水**
- 系統櫃(臥室衣櫃、書房)
- 冷氣 + 安裝
- 燈具 + 開關

### 高規包含什麼

- 標準所有項目
- 設計師全程設計監工
- **動格局**(打牆、加牆、改動線)
- 進口建材(義大利磚、德國衛浴、義大利燈具)
- 客變(玄關、廊道、收納設計)
- 智慧家電整合

## 重點項目 5 大花費明細

我用一個**30 坪標準翻新**為例,把錢都花到哪一一拆解:

### 項目 1:水電管線(5-15 萬)

**最不能省的項目**。

老屋 30 年的水管(鍍鋅鋼管)會生鏽,漏水機率極高;電線老化會跳電,甚至有火災風險。

| 細項 | 預算 |
|------|------|
| 換全屋電線(30 坪) | 3-6 萬 |
| 換全屋水管 + 排水 | 3-6 萬 |
| 加裝專用迴路、無熔絲開關 | 1-2 萬 |
| 弱電(網路、電視) | 5,000-1 萬 |

**全套合計:7-15 萬**。

⚠️ 預算極限時可只換主幹線、保留分支,但**不建議省這個**。

### 項目 2:衛浴廚房(10-25 萬)

**濕區風險最高**。漏水、發霉、防水沒做好,2-3 年後全部都會出問題。

| 細項 | 預算 |
|------|------|
| 衛浴(2 套):防水 + 設備 + 磁磚 | 8-15 萬 |
| 廚房:水槽 + 廚具 + 油煙機 + 排油煙管 | 5-12 萬 |
| 浴缸或淋浴間玻璃 | 1-3 萬 |

**全套合計:14-30 萬**。

⚠️ **防水千萬不能省**,防水做不好,5 年內漏水樓下,賠錢又翻新,得不償失。

### 項目 3:油漆地板(10-20 萬)

**視覺感受最直接**。新油漆 + 新地板,空間就煥然一新。

| 細項 | 預算 |
|------|------|
| 全屋油漆(批土 + 乳膠漆) | 6-10 萬 |
| 拋光地板(超耐磨) | 5-10 萬 |
| 木地板(實木 / 海島型) | 8-15 萬 |
| 磁磚(客廳、玄關) | 4-8 萬 |

**全套合計:10-20 萬**。

### 項目 4:系統櫃 / 木工(10-30 萬)

| 細項 | 預算 |
|------|------|
| 玄關櫃 | 2-5 萬 |
| 主臥衣櫃 | 3-8 萬 |
| 書房櫃 + 書桌 | 3-6 萬 |
| 電視牆 + 矮櫃 | 2-5 萬 |
| 床頭板 | 1-3 萬 |

**全套合計:10-25 萬**。

⚠️ 木工 vs 系統櫃:木工貴 30-50%,但設計彈性高;系統櫃便宜、快、品質穩,適合預算控。

### 項目 5:軟裝家具(另計,30-100 萬)

**軟裝是「裝潢預算外」的另一筆錢**,很多人會漏算。

| 細項 | 預算 |
|------|------|
| 沙發 | 3-8 萬 |
| 餐桌椅 | 2-5 萬 |
| 床墊 + 床架(主臥) | 3-10 萬 |
| 冰箱 + 洗衣機 + 烘衣機 | 5-12 萬 |
| 冷氣(全屋 3 台) | 6-12 萬 |
| 燈具 | 2-5 萬 |
| 窗簾 | 2-5 萬 |
| 飾品、雜物 | 3-10 萬 |

**全套合計:30-70 萬**。

我把上面 5 個項目合在一起:

| 項目 | 預算範圍 | 中位數 |
|------|---------|--------|
| 水電管線 | 7-15 萬 | 10 萬 |

| 油漆地板 | 10-20 萬 | 15 萬 |

| **裝潢總計** | **45-103 萬** | **70 萬** |

## 為什麼一定要留預備金

老屋拆開後**常見的意外**:

1. **牆內水管爛掉**(原本沒打算換,看到爛掉只好補) → 多 3-5 萬
2. **地板下空腐 / 白蟻**(撕掉地板才看到) → 多 2-5 萬
3. **樑柱裂縫 / 鋼筋外露**(批土時才發現) → 多 3-8 萬
4. **磁磚剝離**(拆掉舊磁磚才發現底層鬆動) → 多 2-4 萬
5. **電線比預期老化**(換到一半才發現電盤要換) → 多 1-3 萬

**這些都是「拆了才知道」的意外**。

**預算 80 萬的工程,實際做 88-92 萬很正常**。預備金沒留,你只能中途加碼或縮減其他項目。

## 給讀者的 action items

要翻新中古屋讀完做這 5 件事:

1. ✅ 找 2-3 家工班 / 設計師估價,**估價單必須項目明細齊全**(別只給總價)
2. ✅ 確認估價單有沒有寫「**水電管線換新**」(沒寫表示不換)
3. ✅ 預備金抓總預算的 10-15%
4. ✅ 軟裝家具預算另計,別併入裝潢預算
5. ✅ 簽約前確認**付款方式**(分期付款比一次付清安全)
