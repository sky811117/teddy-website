中部一位朋友前陣子跟我聊到他的客戶 ——

那位客戶買中部一間中古屋,屋主開價 1280 萬,雙方談好。客戶**不想付那 7,680 元的履約保證費**,主張「我認識屋主朋友的朋友,信得過」。

他付了 **訂金 60 萬 + 簽約金 60 萬 = 120 萬**,沒走履保。

結果簽約後第 8 天:

> 屋主**把房子另外抵押給民間借貸公司**,借了 800 萬。

客戶 120 萬訂金 + 簽約金 + 律師費 + 1 年訴訟時間 → 最後只追回 75 萬。**虧 45 萬 + 一年時間**。

**履保費 7,680 元,他省了。**
**45 萬,他賠了。**

但同時我也有過幾位客戶,**用親友交易 + 一次付清,我建議他們可以省履保費**。

履約保證不是「一定要用」也不是「一定不用」。是**看情境**。這篇給你決策框架。

---

## 一句話直接答案

**5 個情境判斷:總價 / 貸款 / 賣方信用 / 抵押權狀態 / 信任程度。任何一個情境踩線,就用履保**。

---

## 情境 1:總價

| 總價 | 建議 |
|------|------|
| < 500 萬 | 看其他情境 |
| 500-1000 萬 | 建議用,履保費可協商 |
| > 1000 萬 | **強烈建議用**,履保費對總價比例極低 |

**邏輯**:履保費是房屋總價的萬分之 6(0.06%)。
- 500 萬房 → 履保費總共 3,000 元(各 1,500)
- 1000 萬房 → 履保費總共 6,000 元(各 3,000)
- 2000 萬房 → 履保費總共 1.2 萬(各 6,000)

**價格越高,履保費佔比越低,但風險越高**。1000 萬以上不用履保 = 賭。

---

## 情境 2:你是否貸款買?

| 你的付款方式 | 建議 |
|------------|------|
| 一次付清(自備款) | 看其他情境 |
| **有貸款** | **必用** |

**邏輯**:
- 貸款買意味著「**銀行先撥款 → 你才有錢付**」
- 中間有 7-14 天時間差
- 如果這段期間賣方違約(把房抵押給第三方、賣給別人),銀行已撥款給你 — 你錢沒了、房沒了、還倒欠銀行貸款

**履保的價值**:把銀行撥款卡在履保專戶,賣方完成過戶才放款給賣方。

---

## 情境 3:賣方信用

| 賣方狀態 | 建議 |
|---------|------|
| 直系親友(父母 / 兄弟姊妹) | 可省 |
| 朋友 / 同事(認識 2 年以上) | 看其他情境 |
| 仲介牽線的陌生屋主(住附近、好查) | 看其他情境 |
| **網路認識的賣家** | **必用** |
| **拍賣公司 / 法拍仲介** | **必用** |
| **建商代售** | 看建商規模、商譽 |

**邏輯**:你**對賣方了解多少**決定風險。

- 直系親友違約幾乎不可能(家族壓力)
- 仲介牽線的本地屋主,住附近、好查、跑不掉
- 網路 / 拍賣公司認識的,**你連賣方真實身分都不一定能確認**

---

## 情境 4:抵押權 / 查封狀態(最關鍵)

| 賣方房子狀態 | 建議 |
|------------|------|
| 完全乾淨(無抵押、無查封、無稅務問題) | 看其他情境 |
| 有抵押權(房貸中) | 看金額 |
| **有抵押權 + 抵押金額接近房價** | **必用** |
| **有強制執行 / 查封紀錄** | **必用 + 諮詢律師** |
| **賣方有訴訟紀錄(被告)** | **必用** |

**怎麼查?**

去**地政事務所**申請「**第二類謄本**」(50 元),可看到:

- 抵押權:金額、債權人、設定日期
- 查封 / 假扣押
- 限制登記

⭐ 這個我每次簽約前**一定先幫客戶查**。

**為什麼這麼重要?**

開頭那位中部客戶的賣方 — 房子原本就有 200 萬抵押,賣 1280 萬可以塗銷後乾淨過戶。但**簽約後屋主再去借 800 萬抵押**,完全違法、但**他在過戶完成前還是房子所有人**,有權設定抵押。

履約保證會把錢卡住,過戶完成才放款給賣方,**屋主就沒誘因再抵押**。

---

## 情境 5:雙方信任程度

| 關係 | 建議 |
|------|------|
| 父母把房子過戶給子女 | 可省 |
| 親兄弟姊妹間轉讓 | 可省 |
| 同事 / 朋友 5 年以上,雙方理解 | 看其他情境 |
| 仲介公司全程處理(雙方都信任仲介) | 看其他情境 |
| **任何一方覺得對方怪怪的** | **必用** |

⚠️ **直覺很重要**。簽約過程中如果你感覺「**屋主怎麼急著趕快收訂金**」「**這個價格怎麼便宜這麼多**」「**屋主說的話前後對不上**」 — **不要省那幾千塊履保費**。

## 我的客戶情境決策矩陣

我會問客戶 5 個是非題:

```
□ 房子總價超過 800 萬?
□ 你有跟銀行貸款?
□ 賣方是陌生人 / 拍賣公司 / 網路認識?
□ 賣方有抵押權尚未塗銷、或有訴訟 / 查封紀錄?
□ 雙方信任不足 / 任何一方覺得有疑慮?
```

**任一個 → 用履保**
**全部都沒有 → 可考慮省**

## 兩個真實對比

### 案 A:用了履保,平安落幕(我去年客戶)

- 1450 萬中古屋
- 有貸款 + 賣方原本有 300 萬抵押
- 履保費總共 8,700 元(雙方各 4,350)
- 過戶過程中銀行查到賣方有舊訴訟未結
- 履保專戶卡住,我幫客戶協調賣方先處理訴訟、再過戶
- **如果沒用履保**:訴訟方可能突然來查封,客戶卡死

### 案 B:省履保費沒事(同事的親友交易)

- 父親把房子過戶給女兒,1080 萬
- 雙方對房子狀況都熟、無抵押、無問題
- 省下 6,500 元
- **0 風險**
