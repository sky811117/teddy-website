上週帶一位中部朋友看了 3 間房子。

她是第一次買房,看完第三間我們在咖啡廳坐下,她說:

> 「景泰,我才知道原來『帶看』不是『去看 30 分鐘就完事』。」

我問她以前怎麼想的。她說:「就上 591 找房子,約仲介開門看一下,30 分鐘走完，回家想要不要買。」

**那其實是看屋,不是帶看**。

買房不是去逛 IKEA。一間房 1000 萬起跳,如果只看 30 分鐘就要決定,那不是太草率了嗎?

這篇把我帶看完整流程攤開來給你看。

---

## 一句話直接答案

完整帶看分 4 階段:**配對 → 約看 → 現場 → 看後 3 天**。一個負責任的房仲不只是「開門」,他會在 4 個階段全程陪你。

---

## 階段 1:配對(看屋前 1-7 天)

這是**最被低估的階段**。

90% 買方不知道:**仲介在你看屋前,可能已經幫你篩掉 20 間了**。

### 我會做的事

1. **問清楚需求**:
   - 預算上限(總價 + 自備款)
   - 區域(北屯、西屯、南屯、市中心?)
   - 坪數 / 房數
   - 屋齡(中古、新成、預售)
   - 樓層(避開頂樓、低樓層、4 樓?)
   - 不能接受的條件(車位、嫌惡設施、無電梯)

2. **從我手上 200+ 物件中篩**:
   - 公司聯賣系統
   - 我自己經營的物件
   - 同業推薦(透過聯賣)

3. **每間都先做功課**:
   - 上實價登錄查同社區成交價
   - 看不動產說明書有沒有揭露
   - Google Map 掃周邊嫌惡設施
   - 屋齡 / 樓層 / 公設比

4. **挑出 3-5 間配對度最高的**,LINE 給你看,確認你想看哪幾間

⚠️ **重點**:**仲介推薦你看的物件 = 他已經幫你過濾過了**。但如果你跟仲介不熟、需求沒講清楚,他可能只是把手上要清的物件硬塞給你。**問清楚需求 + 給仲介充分資訊**很重要。

---

## 階段 2:約看(看屋前 1-3 天)

### 我會處理的事

1. **跟屋主協調時間**
   - 自住屋:屋主通常週末才方便(平日上班)
   - 租賃屋:看房客作息,一般要提前 3 天約
   - 空屋:彈性最大,隨時可看

2. **抓好間距**
   - 一個下午看 2-3 間最理想
   - **不要排超過 4 間** — 看到後面記憶會糊掉
   - 每間之間留 30 分鐘車程 + 緩衝

3. **準備資料夾**
   - 不動產說明書影本
   - 實價登錄查詢頁
   - 嫌惡設施 Google Map 截圖
   - 我的屋況檢查表

4. **提醒買方該準備什麼**
   - 帶筆 + 筆記本(寫感想)
   - 帶量尺(量房間放不放得下你的家具)
   - 帶長輩或家人一起去(複看必備)
   - 穿好脫的鞋(很多屋主要求脫鞋)

## 階段 3:現場(看屋當天 — 30-60 分鐘 / 間)

### 我會帶的東西

| 物品 | 用途 |
|------|------|
| 不動產說明書影本 | 對照屋主說的、跟說明書是否一致 |
| 謄本影本 | 確認所有人、貸款餘額、抵押 |
| 量尺(雷射或捲尺) | 量主臥、客廳、廚房尺寸 |
| 手電筒 | 看天花板水痕、衣櫥裡面 |
| Google Map 截圖 | 周邊嫌惡設施 |
| 屋況檢查表 | 一張紙,12 個項目 check |
| 計算機 | 算單坪 + 自備款試算 |
| 名片 + 委託書 | 屋主或代理人不見得認識我 |

### 現場 60 分鐘怎麼分配

**0-10 分鐘:外觀 + 公設**
- 大樓外觀有沒有滲水、磁磚剝落
- 大廳、信箱區、垃圾室、健身房(如果有)
- 樓梯 / 電梯狀況

**10-25 分鐘:室內格局**
- 進門順序走一輪(玄關 → 客廳 → 廚房 → 衛浴 → 房間)
- 每個空間量尺寸
- **打開所有櫥櫃**(看內部、看有沒有裝潢藏東西)
- 試試所有水龍頭、馬桶沖水、瓦斯爐(屋主同意的話)

**25-45 分鐘:屋況檢查**
- 天花板水痕(浴室上方、陽台旁)
- 牆角壁癌(尤其角落、衣櫥後面)
- 窗戶能不能完全關上(膠條老化)
- 漏水痕(浴室矽利康、廚房水槽下)
- 噪音(打開窗 5 分鐘聽外面)
- 屋齡 vs 屋況落差(老屋如果整理得太新,要懷疑漏水重新油漆)

**45-60 分鐘:跟我討論**
- 你的第一印象
- 我看到的優缺點
- 跟其他案件比的位置
- 你想下一步做什麼

## 階段 4:看後 3 天(我做的事)

很多買方以為「**看完就結束了**」。其實看完才是開始。

### 看後 24 小時內

- 整理今天看的物件**對照表**(下面範例)
- 找實價登錄同社區、同樓層、近 1 年成交價
- 算合理單坪價
- 列出**優點 / 缺點 / 待確認**清單

### 看後 48-72 小時

- LINE 給你**完整評估報告**:
  - 市場行情比對
  - 屋主開價 vs 合理價的差距
  - 議價策略(可以下多少、怎麼下)
  - 風險提醒(漏水、糾紛、嫌惡設施)

- 如果你想再看一次,**約屋主複看**

## 買方最常犯的 3 個錯

### 錯誤 1:只看 1 間就決定

「我感覺對了」這 4 個字毀掉很多人的買房經驗。

**至少看 5 間以上才開始做選擇**。你不看比較組,根本不知道什麼叫合理價。

### 錯誤 2:沒帶長輩 / 家人複看

第一次看自己看就好,但下訂前**一定要找另一個人複看**。

- 長輩看風水、屋況(他們經驗多)
- 配偶看格局實用性
- 朋友看冷靜的第二意見

### 錯誤 3:不問漏水 / 改建 / 糾紛

「不好意思」3 個字讓很多買方吃虧。**買房 1000 萬,你怎麼可以不好意思問**?

該問的:
- 這間以前有沒有漏過水?
- 有沒有改建 / 違建?
- 樓上鄰居好嗎?
- 為什麼屋主要賣?
- 已經放多久了?

不會有人因為你問而生氣 — 真的會生氣的,可能就是有事要藏。

## 你該做的 4 件事

1. **看屋前**把需求講清楚給仲介,別讓他亂推
2. **看屋當天**帶量尺、手電筒、長輩或家人
3. **每間至少看 2 次**才下訂(白天 + 傍晚)
4. **看後 3 天**等仲介給你完整評估再決定
